# Kang

Small collaborative kanban board.

## Workspace

- `apps/api` contains the Bun and Elysia backend.
- `apps/web` contains the React and Vite frontend.

## Commands

```bash
bun run dev:api
bun run dev:web
```

Other useful commands:

```bash
bun run build:web
bun run lint
```

## Deployment

The plan is for it to be deployable via Docker/Podman Compose.
